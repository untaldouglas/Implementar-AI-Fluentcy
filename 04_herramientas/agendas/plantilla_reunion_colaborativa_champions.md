# Plantilla — Reunión de Trabajo Colaborativo entre Champions

| Campo | Valor |
|---|---|
| Frecuencia | Semanal (resto de julio 2026 — Fase 1) |
| Duración | 2 horas |
| Participantes | Irvin Morales · Mario Valencia · Patrick Orellana |
| Facilitador | Rotativo entre Champions (definir por sesión) |
| Objetivo | Compartir avances y construir conocimiento colectivo sobre uso de Hermes Agent, sin la Dirección presente |

---

## Estructura sugerida (2h)

| Bloque | Tiempo | Contenido |
|---|---|---|
| Check-in | 10 min | Cada Champion comparte en una frase cómo le fue con Hermes esta semana |
| Demo / caso compartido | 40 min | Cada Champion muestra un caso de uso, prompt o solución que descubrió (rotativo: ~12–13 min c/u) |
| Trabajo colaborativo en vivo | 50 min | Resolver juntos un bloqueo real de alguno de los tres (pair-work), o construir un caso de uso conjunto entre áreas |
| Documentación | 10 min | Registrar en el `learning_record` de cada uno lo aprendido/resuelto hoy |
| Cierre y acuerdos | 10 min | ¿Qué se lleva cada uno para practicar esta semana? ¿Qué tema se propone para la próxima sesión? |

## Notas de uso

- Esta reunión es **entre pares**, sin Dirección — el espacio es para que los Champions construyan confianza y conocimiento horizontal.
- Si surge un bloqueo que requiere decisión de Dirección, se anota aquí y se lleva a la Reunión de Seguimiento del Proyecto (ver `plantilla_reunion_seguimiento_proyecto.md`).
