# Cuestionario Baseline — AI Literacy
**Dirección de Servicios Informáticos · UJMD**
**Versión:** 1.0 | **Tiempo estimado:** 25–30 minutos
**Aplicar a:** 3 Champions seleccionados (1 por área) + extender al equipo completo en Fase 2

---

## Instrucciones para el facilitador (Douglas)

**Propósito:** Diagnosticar el punto de partida de cada participante en 4 dimensiones antes del piloto:
1. Herramientas de IA que usa actualmente
2. Casos de uso reales donde aplica IA
3. Nivel de autonomía alcanzado (mapeable a L0–L9 del Manual)
4. Bloqueos percibidos (técnicos, confianza, tiempo, éticos)

**Formato de aplicación:** Entrevista semi-estructurada (presencial o videollamada) o auto-administrado con seguimiento. Se recomienda la primera opción para Champions: las respuestas abiertas dan más señal que las cerradas.

**Resultado esperado:** Ficha individual con nivel actual estimado + 3 casos de uso priorizados + bloqueos dominantes. Insumo para diseñar el itinerario personalizado de cada Champion dentro del piloto.

| A10 | He programado agentes que trabajan solos en horario no laborable (auditorías, reportes) | | | | | | L8 |

### Cálculo de nivel estimado

- Suma de puntos A1–A10 = ______
- Mayor puntuación en un nivel = **nivel dominante estimado**
- Interpreta:
  - **L0–L2**: Zona del Pasajero — necesita nivelación en fundamentos
  - **L3–L5**: Zona del Conductor — listo para construir Skills y MCP
  - **L6–L8**: Zona del Piloto Automático — puede ser mentor de otros
  - **L9**: Nivel visionario — raro; confirmar con evidencia real

## Anexos

### A. Tabla de mapeo rápido: respuesta → nivel

Si la respuesta en Sección 4 tiene 4–5 principalmente en niveles bajos (A1–A4) → **Pasajero**.
Si tiene 4–5 en niveles medios (A3–A7) → **Conductor**.
Si tiene 4–5 en niveles altos (A7–A10) → **Piloto Automático**.

### B. Criterios para selección final de Champions

El Roadmap dice: *"no el más avanzado ni el más novato, sino el de mayor apertura al cambio y capacidad de transferir conocimiento"*. Cruza:
1. Nivel estimado (Sección 4): ideal L2–L4 (ni muy novato ni muy experto)
2. Bloqueo "cultura" bajo (Sección 5): señal de apertura
3. Respuesta afirmativa a "¿enseñarías a tu área?" (Sección 6)
4. Señales del facilitador (Sección 7)
